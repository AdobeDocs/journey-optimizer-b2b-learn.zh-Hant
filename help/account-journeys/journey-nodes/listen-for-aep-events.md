# 接聽AEP事件

在[!DNL Journey Optimizer B2B Edition]中接聽AEP事件可讓您定義並使用帳戶歷程中的任何體驗事件。 管理員可以設定Adobe Experience Platform (AEP)型事件定義，讓行銷人員建立可對AEP Experience事件回應的帳戶歷程。 在帳戶歷程中使用AEP體驗事件的程式分為兩個部分：

* 建立及發佈AEP事件定義

* 在帳戶歷程中使用AEP事件

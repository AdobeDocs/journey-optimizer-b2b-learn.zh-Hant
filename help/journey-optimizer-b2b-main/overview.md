# [!DNL Journey Optimizer B2B Edition] 教學課程

了解如何充分利用 [!DNL Journey Optimizer B2B Edition]。使用內建的生成式 AI 和業界領先的自動化功能，協調帳戶與購買群組歷程，最大限度地提高對特定方案的需求。

>

* [建立帳戶歷程](/help/account-journeys/create-an-account-journey.md)
